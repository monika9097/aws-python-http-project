Hi, I Built This Project From Scratch!

I deployed my application in one of the private subnets and accessed it using a bastion host. This setup helped me understand how a Load Balancer works by routing traffic to a subnet where the app is running. I created a VPC with two private subnets and one public subnet, along with an Auto Scaling group having two instances — one per Availability Zone. Since private subnets are per AZ, the bastion host was created in the public subnet. I copied my EKS key to one of the private subnet instances, SSHed to it using the bastion host, and deployed a simple Python app with this index.html file. Finally, I configured an Application Load Balancer to send traffic to the working private subnet instance.

Made with ❤️ using AWS EC2, VPC, Load Balancer, and Python HTTP Server
